🇫🇷 Français Master Pro | 法语数字听写特训
一个现代化的、交互式的网页应用，旨在帮助法语初学者掌握 0 到 100 的法语数字听写与拼写。单文件设计，无需后端，下载即用。

✨ 主要功能
双重练习模式：

顺序闯关 (Sequential)：从 0 到 100 循序渐进地练习。

随机抽查 (Random)：自定义数字范围（例如 20-60），系统随机出题，并在完成后提供统计。

原生语音支持：利用浏览器自带的 TTS (Text-to-Speech) 引擎自动朗读法语数字，支持点击重读。

智能纠错反馈：

输入错误时会有震动提示。

逐字符比对，绿色高亮正确部分，红色标记错误部分。

错题本 (Mistake Log)：自动记录答错的数字及次数，方便针对性复习。

辅助输入：内置特殊字符软键盘（é, è, à, ç, -），无需切换系统输入法。

现代化 UI：基于 Tailwind CSS 构建的毛玻璃 (Glassmorphism) 风格界面，适配移动端，包含撒花庆祝动画。

🛠️ 技术栈
核心：原生 HTML5 / JavaScript (ES6+)

样式：Tailwind CSS (CDN 引入)

图标：FontAwesome

特效：Canvas Confetti

🚀 如何使用
下载 number.html 文件。

直接使用浏览器（Chrome/Edge/Safari）打开该文件即可开始练习。

无需安装 Node.js 或任何依赖。
